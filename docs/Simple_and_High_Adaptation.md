# 简单，适配度高

和面向家用单反一样，如果要实现简单，适配度高，需要在代码的逻辑上增强对相机的识别和协议的识别，具体的流程如下

```
+---------------------------------------------------------+
|                  auto|detect connection                 |
+----------------------------+----------------------------+
                             |
                             |
+----------------------------v----------------------------+
|                 get usb vendor and device id            |
+----------------------------+----------------------------+
                             |
                             |
+----------------------------v----------------------------+
|         Search local(remote) to get best protocol       |
+----------------------------+----------------------------+
                             |
                             |
+----------------------------v----------------------------+
|            Start sync(use ptp or wifi or msc or others) |
+------------+-----------------------------------+--------+
             |                                   ^
             |                                   |
             |                                   |
             |                                   |
             +-----------------------------------+

if failed , auto change to other protocol and report to server
```
